# PJV – CTU Programming in Java

This repository contains my coursework, lab work, and programming assignments for the CTU Programming in Java course.

## About

I keep this repository public mainly as a personal archive of my work and as a study resource for other students.  
It includes lab folders, exercises, homework tasks, and smaller course-related projects created during the semester.

- homework tasks
